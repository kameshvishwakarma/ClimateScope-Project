# ClimateScope Project 🌦️

## 📌 Project Overview

ClimateScope is a weather data analysis project that helps in understanding climate patterns using data visualization and statistical analysis.

## 📊 Features

* Weather data analysis (Temperature, Humidity, Visibility)
* Interactive dashboard (Power BI)
* Data cleaning and preprocessing
* Visualization using charts and graphs

## 🛠 Tools & Technologies

* Python (Pandas, NumPy, Matplotlib, Seaborn)
* Power BI
* Jupyter Notebook

## 📁 Files in Project

* `clean_data.csv` → Dataset
* `Processes_data.ipynb` → Data analysis code
* `finaldashboard.pbix` → Dashboard
* `design documentation.pdf` → Project design

## ▶️ How to Use

1. Open Jupyter Notebook
2. Run `Processes_data.ipynb`
3. Open Power BI file to view dashboard
